Contexto do Projeto

O controle de pesagem do rebanho é essencial para a qualidade de vida dos animais e a lucratividade das propriedades rurais. Monitorar o peso de bovinos, equinos e outros animais permite avaliar o desempenho no ganho de peso, ajustar a alimentação e identificar precocemente problemas de saúde.

Atualmente, a maioria das propriedades realiza esse controle manualmente, o que gera desafios como necessidade de mão de obra especializada, estresse nos animais e possibilidade de erros humanos.

Nossa Solução

Sistema de Pesagem Automatizada

Utilizamos tecnologia NFC para identificação individual dos animais e um mecanismo de controle com portões automatizados que garantem que apenas um animal seja pesado por vez.

Gestão de Dados Inteligente

Todos os dados são armazenados em banco de dados MySQL e acessíveis via aplicação web com dashboard intuitivo, médias de peso, alertas de anomalias e histórico completo.

Chat com Inteligência Artificial

Nosso sistema inclui um assistente virtual que permite interagir com os dados de forma natural, sem necessidade de conhecimento técnico. Basta perguntar e receberá as informações que precisa sobre seu rebanho.

Benefícios do Sistema

Redução de erros

Minimiza erros humanos no registro de pesos

Eficiência operacional

Reduz necessidade de mão de obra especializada

Monitoramento contínuo

Identifica variações de peso que indicam problemas

Menor estresse animal

Processo automatizado reduz desconforto nos animais

Alertas instantâneos

Exibe anomalias no desenvolvimento das pesagens

Acesso remoto

Dados disponíveis em tempo real de qualquer lugar

Tecnologias Utilizadas

Hardware

  • Microcontrolador ESP32
  • Sensor NFC RFID-RC522
  • Células de carga strain gauge
  • Módulo amplificador HX711
  • Servo motores para portões

Software

  • Flask (Backend Web)
  • MySQL (Banco de dados)
  • MQTT (Comunicação)
  • Tailwind CSS (Interface)
  • LLM (Chat Inteligente)
